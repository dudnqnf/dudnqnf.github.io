## 개요

* 근무처 : Mapitizer(맵피타이저)
* 발주처 : Mapitizer
* 역할 : 안드로이드 개발
* 앱 설명 : 모바일 청첩장 기능, 개인 웨딩룸 꾸미기
* 개발상세내역 :
* 앱에 필요한 기본적인 서비스구현
* 페이스북, 카카오톡, 네이버, 구글 로그인 api
* 네이버 지도api
* URL : https://play.google.com/store/apps/details?id=com.takebox.wedding
* 비고 : 앱 등록 후 서비스 중
